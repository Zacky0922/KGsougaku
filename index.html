HTMLの基本

HTML＝Hyper Text Markup Language（ハイパー・テキスト・マークアップ・ランゲージ）とは、文章をマークアップ（＝タグで囲うこと）して機能を持たせる、ウェブサイトの表示に適した形式の文章のことです。

テキストの表示

テキストには、大きく分けてブロック要素・インライン要素の2種類があります。
段落のような改行を伴う塊を作る場合はブロック要素、改行せず行内の一部を指定する場合はインライン要素を使用しましょう。
ブロック要素
pタグ
段落(paragraph)を表す。
divタグ
分割された区間（division）を表す。
インライン要素
spanタグ
テキストの一部を装飾するときに用いる。
aタグ
テキストにリンク等の機能を持たせるときに用いる。 
<a href="リンク先URL" target="リンクの方法">表示するテキスト</a>

表の表示

表は、大きく次の3要素で成り立っています。
  • 表の大枠 tableタグ
  • 行 trタグ
  • セル（マス目） tdタグ
この3つを意識して、書きましょう。 
<table>
    <tr>
        <td>
            A
        </td>
        <td>
            B
        </td>
        <td>
            C
        </td>
    </tr>
    <tr>
        <td>
            D
        </td>
        <td colspan="2">
            ここは2つ合体する
        </td>
    </tr>
</table>
A B C
D ここは2つ合体する

CSSの準備（IDとクラス）

要素に「固有の名称＝ID」や、「グループの名称＝クラス」を設定することができます。
CSSで装飾を設定する際に、まとめて設定するときに便利です。
同じIDは、1つのページ内に1つしか設定することができませんが、クラスは複数設定することができます。

CSSの基本

CSS＝Cascading Style Sheets（カスケーディング・スタイル・シート）とは、HTMLで記載された文章を装飾するための命令のことです。

指定の方法

原則、「適用対象{ 項目：内容; }」の形式で指定します。
また、指定を書く方法は、大きく3種類あります。
直接指定する
個別に指定したい場合に利用します。 
<p>
みなさんこんにちは。私の名前は<span style="color:#f00;">国際太郎</span>です。
</p>

みなさんこんにちは。私の名前は国際太郎です。

まとめて指定（ページ内で）
ページ内全体に適用したい場合に利用します。
同一の要素に対し、複数の指定がある場合は、「より細かく指定した」ものが適用されます。 
<style>
span{
    color: #f00;
    font-size: large;
}
span.blue{
    color: #00f;
}
</style>
<p>
<span>今日の天気は
<span class="blue">雨</span>
です。
</span>
</p>

今日の天気は雨です。

まとめて指定（ページ外で）
複数のページに適用したい場合に利用します。
HTMLファイルとは別に、CSS（～～.css）ファイルを用意し、その中で設定します。 
<html>
<head>
<link rel="stylesheet" href="CSSファイルのURL">
</head>
<body>
（本文）
</body>
</html>

代表的な装飾

位置設定系
margin
要素外部の余白設定です。
指定する数値の数によって、余白を指定する位置を変更できます。
  • margin: [上下左右];
  • margin: [上下] [左右];
  • margin: [上] [左右] [下];
  • margin: [上] [下] [左] [右];
padding
要素内部の余白設定です。
指定する数値の数によって、余白を指定する位置を変更できます。
（marginと同様）
フォント・罫線
color
文字色を設定できます。
font-size
文字の大きさを指定します。
数値の他に、「xx-small、x-small、small、medium、large、x-large、xx-large」などが指定できます。
border
罫線についての設定です。
太さ、種類、色の3つを指定します。（順序は自由）
border:1px solid #f00; border:5px dashed blue; border:3px dotted rgb(0,255,0);
